## 👋 参加编辑
> **‼️ 编辑须知**
> 
> 你需要知道 **关于 *Minecraft 资源包*的制作知识** ，并会使用 **Git** 。
首先，你需要 **Fork** 此仓库。

其次，完成编辑。

最后，请在你的仓库里点击按钮开启 **Pull Requests** ，等待审核通过。

通过之后，你的修改就被合并到主分支啦。
